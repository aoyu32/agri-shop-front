# 农购宝 - 智能农产品产销服务系统

基于 Vue 3 + Element Plus 的智能农产品产销服务平台前端项目。

## 功能特性

- ✅ 基础布局系统（Header + Footer + Main）
- ✅ 多角色支持（消费者、商户、管理员）
- ✅ 响应式头部导航栏
- ✅ 用户状态管理（Pinia）
- ✅ 路由系统（Vue Router）
- ✅ 绿色农产品主题

## 技术栈

- Vue 3
- Vite
- Element Plus
- Vue Router
- Pinia
- Sass

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 角色说明

### 消费者（Consumer）
- 导航栏：首页、分类、AI咨询、社区
- 功能：购物车、订单、消息、个人中心

### 商户（Merchant）
- 导航栏：首页、分类、AI咨询、社区、**行情预测**
- 功能：订单、消息、我的店铺、个人中心

### 管理员（Admin）
- 待开发

## 开发说明

### 角色切换
在首页点击"切换为消费者"或"切换为商户"按钮可以切换角色，查看不同角色的界面差异。

### 项目结构
```
src/
├── assets/          # 静态资源（字体、图标）
├── components/      # 公共组件
├── router/          # 路由配置
├── store/           # 状态管理
│   └── modules/     # 状态模块
├── styles/          # 全局样式
├── view/            # 页面视图
│   └── layout/      # 布局组件
└── main.js          # 入口文件
```
